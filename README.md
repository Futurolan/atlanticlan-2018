# Gamers Assemby Front
This repository contain the Gamers Assembly website frontend.

It's build on fun technologies like:
- React Nextjs (https://github.com/zeit/next.js)
- Graphql (http://graphql.org/)

## Installation
```
npm install 
```

## Dev 
```
npm dev
```

## Production 
```
npm build 
npm start 
```
